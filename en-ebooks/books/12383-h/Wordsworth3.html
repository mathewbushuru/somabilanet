The Wordsworth Series

In order to take advantage of the cross-reference links between volumes in this series, you need to:
  1. download the volumes (1-8), as they become available
  2. rename each volume folder (if they are not named this way): Wordsworth1, Wordsworth2 etc.
  3. rename the main file in each volume (if not named this way): Wordsworth1.html, Wordsworth2.html etc.
  4. rename the other html files in each volume (if not named this way): Wordsworth 1a.html, Wordsworth1b.html etc., Wordsworth2a.html, Wordsworth2b.html etc.
As the series is completed, the cross-references between volumes will be updated, so make sure you have the most recent version of this download from Project Gutenberg.
